# Level 1 Scope — AquaTrade Pay

## Goal

Ship a working public Stellar Testnet dApp that connects Freighter, displays XLM balance, and sends XLM.

## In scope

- Freighter setup guidance
- Connect and disconnect
- Testnet guard
- XLM balance
- Friendbot funding
- Direct Testnet XLM payment
- Success/failure state
- Transaction hash and explorer link
- Docker setup, tests, public deployment, README, screenshots

## Out of scope

- Backend API dependency
- Database
- Worker
- Smart contract
- Escrow
- Mainnet
- Real-money settlement
- Full export-import workflow
